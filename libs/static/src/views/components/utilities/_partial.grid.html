Grid

FDS Design System uses mobile-first flexbox grid for all sizes. The grid contains twelve columns, five default responsive breakpoints, and couple of predefined utility classes. The flex grid is only supported in Chrome, Firefox, Safari 6+, IE10+, iOS 7+, and Android 4.4+.

Flex grid uses row class ( .row ), and column class (.col ). Basic percentage-based sizing can also be done using the classes: .col-sm-6, .col-md-6, and so on.
